# Hotel Operations Data Analysis Dashboard

## Overview
This project analyzes operational hotel data to understand occupancy trends, guest behavior, and breakfast participation.

The dataset was collected during **February** and includes guest counts, breakfast consumption, room availability, and guest segmentation.

## Dataset
The dataset tracks daily operational metrics such as:

- Total guests
- Guests who had breakfast
- Guests who did not have breakfast
- Total rooms
- Occupied rooms
- Available rooms
- Guest type
- Adults vs minors

Total analyzed:
- **981 guests**
- **89.6% breakfast participation**
- **47.48% average occupancy rate**

## Tools Used

- Google Sheets (data collection and KPI calculations)
- Looker Studio (dashboard and visualization)

## KPIs Analyzed

- Occupancy Rate
- Breakfast Participation Rate
- Guest Volume
- Guest Segmentation (Adults / Minors)
- Monthly Operational Trends

# 🇪🇸 Español

## Descripción general

Este proyecto analiza datos operativos de un hotel para comprender tendencias de ocupación, comportamiento de los huéspedes y participación en el servicio de desayuno.

El conjunto de datos fue recolectado durante **todo el mes de febrero** e incluye información sobre cantidad de huéspedes, consumo de desayuno, disponibilidad de habitaciones y segmentación de clientes.

## Dataset

El dataset registra métricas operativas diarias como:

- Cantidad total de huéspedes
- Huéspedes que consumieron desayuno
- Huéspedes que no consumieron desayuno
- Habitaciones totales
- Habitaciones ocupadas
- Habitaciones disponibles
- Perfil de huésped
- Adultos y menores

Totales analizados:

- **981 huéspedes**
- **89.6% de participación en desayuno**
- **47.48% de ocupación promedio**

## Herramientas utilizadas

- Google Sheets (recolección de datos y cálculo de KPIs)
- Looker Studio (dashboard y visualización de datos)

## KPIs analizados

- Tasa de ocupación
- Tasa de participación en desayuno
- Volumen de huéspedes
- Segmentación de huéspedes (Adultos / Menores)
- Tendencias operativas mensuales
